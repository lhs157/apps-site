## FAQ

**Is my data safe?**
Yes. All data is stored locally on your device, encrypted with AES-256 (SQLCipher). Xu has no servers and cannot access your data.

**Do I need an internet connection?**
No. Xu works fully offline. Internet is only needed for optional features: Google Drive backup and Gemini AI (if you choose to use them).

**How do I back up my data?**
Go to Settings → Cloud Backup → sign in with Google → Upload. The backup file is encrypted and stored in your personal Google Drive.

**How do I restore from a backup?**
Go to Settings → Cloud Backup → sign in with Google → select a backup file → Restore.

**What happens if I forget my PIN?**
For security, there is no way to recover a forgotten PIN. Please keep it safe. You can enable Face ID / fingerprint as an alternative unlock method.

**Does the AI use my data?**
The AI assistant runs offline by default using a local rule-based engine. If you choose to enable Gemini AI, text is sent directly to Google's API — Xu does not store or see your conversations.
